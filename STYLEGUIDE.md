# Style Guide

## Plugins

-   ESLint - linter
-   Prettier - formatter

## Editor Intergration

-   Install VS Code ESLint plugin.

## Format Code

    npm run format

## Lint Code

    npm run lint

## Rules

-   Tab Width: 4 Spaces
-   Always use Semicolons
-   Use CamelCase
-   Use Single Quotes for strings
